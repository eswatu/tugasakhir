Cara Penggunaan Aplikasi

Informasi Umum

Form Input

Untuk setiap form input validasi dilakukan secara otomatis, apabila tombol simpan/buat tidak tersedia berarti data masukan tidak valid. Lihat keterangan kebutuhan data di setiap form dengan hover di kolom isian.

Input Target Pada Kontrak Kinerja

  1. Masuk ke Menu "Angka Kredit", lalu pilih tab "Kontrak Kinerja"
  2. Klik tombol "Tambah Target" lalu isi data kontrak kinerja. Untuk tahun masukkan misalnya 2022
  3. Klik "Buat Baru" untuk menyimpan
  4. Setiap Tahun hanya bisa menggunakan 1 kontrak kinerja, ubah kontrak tahun berjalan apabila diperlukan
  5. Dashboard merupakan menu untuk melihat capaian kinerja pada tahun berjalan, otomatis mengikuti tahun terpilih

Input Data Surat Tugas

  1. Masuk ke Menu "Angka Kredit", lalu pilih tab "Surat Tugas"
  2. Klik Tombol "Tambah Baru", tunggu hingga muncul dialog
  3. Masukkan data surat tugas sampai tervalidasi (tombol simpan dapat diklik) lalu klik "Simpan"
  4. Setelah mendapatkan konfirmasi sudah tersimpan, lihat pada tabel surat tugas yang baru berwarna merah tanda beluma memiliki berkas/softcopy
  5. Klik tombol bergambar awan untuk menggunggah file Surat Tugas
  6. Pilih file yang diupload, format PNG, JPG, JPEG, PDF dengan maksimal ukuran 2 Mb dan masukkan keterangan pada kolom "Catatan" (apabila ada)
  7. Klik Upload, setelah selesai dialog dapat ditutup. Untuk file Surat tugas hanya bisa diuggah 1 file
  8. Setelah surat tugas berwarna putih maka dapat digunakan untuk input Hasil Pekerjaan

Input Bukti Pekerjaan

  1. Masuk ke Menu "Angka Kredit", lalu pilih tab "Pekerjaan"
  2. Klik Tombol "Tambah Laporan", akan terdapat pilihan Jenis laporan untuk "Pekerjaan Utama" atau "Pekerjaan Tambahan"
  3. Pilih butir pekerjaan dari pohon pekerjaan hingga muncul form input pekerjaan
  4. Input Data Surat Tugas (dari pilihan), jumlah kegiatan, tanggal pelaksanaan, dan catatan pekerjaan. Lalu klik "Buat Data"
  5. Muat ulang laman, atau klik refresh
  6. Kegiatan yang baru saja diinput akan berwarna merah dan tidak dapat diajukan, klik gambar upload untuk menggunggah bukti pekerjaan/laporan
  7. Pilih file yang diupload, format PNG, JPG, JPEG, PDF dengan maksimal ukuran 2 Mb dan masukkan keterangan pada kolom "Catatan" (apabila ada)
  8. Klik Upload, setelah selesai dialog dapat ditutup. Untuk file hasil pekerjaan bisa lebih dari 1 file.
  9. Apabila baris kegiatan sudah berwarna putih dan terdapat tombol "Ajukan Berkas" maka berkas siap diajukan

Input Pengajuan

  1. Masuk ke Menu "Angka Kredit", lalu pilih tab "Pengajuan"
  2. Klik Tombol "Buat Pengajuan", masukkan nomor/nama pengajuan dan tanggal serta catatan jika diperlukan
  3. Pengajuan awal akan berwarna putih dan terdapat tombol "Ajukan". lanjutkan ke proses Pengajuan
  4. Pengajuan Aktif hanya ada satu, kirim pengajuan dan tunggu validasi apabila ingin membuat pengajuan lain
  5. Manfaatkan menu edit untuk mengubah data Pengajuan

Proses Pengajuan

Pengajuan hasil pekerjaan dilakukan dengan melakukan klik tombol Kirim pengajuan di menu Pekerjaan. Untuk setiap butir Pekerjaan dengan tombol bergambar "Kirim" apabila diklik secara otomatis akan diajukan ke Pengajuan aktif. Nantinya di bagian pengajuan akan muncul hasil pekerjaan yang dikelompokkanm berdasarkan surat penugasan. Setelah seluruh pekerjaan sudah diajukan, kembali ke menu Pengajuan dan klik "Ajukan" untuk mengajukan penilaian, tunggu validasi tim penilai.
